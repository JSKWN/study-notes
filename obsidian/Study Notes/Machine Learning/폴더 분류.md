## 폴더 분류 방법
---

| 대분류          | 중분류                   | 예시 키워드                                             |
| ------------ | --------------------- | -------------------------------------------------- |
| **Model**    | Base Architecture | Transformer, CNN, Diffusion, RNN, Mamba            |
|              | Components        | Attention, LayerNorm, GELU, Residual Connection    |
| **Training** | Paradigm          | Supervised, Self-Supervised, RL, Generative        |
|              | Strategy          | Transfer Learning, Fine-tuning, LoRA, Distillation |
|              | Optimization      | AdamW, Cosine Annealing, Warmup, Loss Function     |
|              | Regularization    | Dropout, Data Augmentation (MixUp)                 |
| **System**   | Efficiency        | Quantization, Pruning, Compression                 |

## 태그 표시
---
태그의 경우 사용분야(모달리티)를 기준으로 아래와 같이 정할 예정
(주로 'CV', 'RL', 'NLP', 'Multimodal' 4개의 태그를 이용하게 될 것 같음)

- **모달리티(Modality - 분야):**
    - **Computer Vision (CV):** 이미지, 비디오 처리
    - **Natural Language Processing (NLP):** 텍스트, 언어 모델
    - **Audio/Speech:** 음성 인식, 합성, 음악 생성
    - **Multimodal:** 이미지+텍스트 (예: CLIP, LLaVA)
    - **Reinforcement Learning (RL):** 로봇 제어, 게임, 의사결정
